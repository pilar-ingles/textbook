HTML - Теги для форматирования текста

У всех тегов может быть необязательны атрибут id=""–идентификатор, который необходим для унификации тега при необходимости

Тег Формат Атрибуты Описание Варианты Emmet
h1...h6 <h1 align="left"> Содержание </h1> align=«center»/"right"/"left" показывает, где находится заголовок «center» - по центру, "right" - справа, "left"-слева Например: <h1 align="left"> Наименование </h1> Теги заголовков, парные, шесть уровней. В браузере заголовки выводятся жирным шрифтом. Разные уровни отличаются размером шрифта. h1-первый уровень, самый крупный. Используется для описания назначения страницы. h2...h6-используются при необходимости. h6*5+Tab-создание 5 заголовков 6 уровня
p <p>Текст</p> <i> (накл. шрифт)</i> , <b> (полужирный) </b>, <sub>(нижн.инд) <sub>, <sup>(верх.инд)<sup>, strong и em служать для выделения голосом при чтении (аналогично тегам b и i). Тег параграфа,парный,оформляет абзац текст. p*5+Tab-создание 5 параграфов
ul <ul><li>......</li></ul> type="square" - будет квадрат Тег неупорядоченного списка, парный. Перед элементами списка точка по умолчанию. li-определяеи элементы списка, количество неограничено. Других тегов после ul быть не может. Но внутри li можно размещать другие. ul>li*5+Tab-создание списка из 5 строк
ol <ol> <li> ... </li> </ol> Тег упорядоченного списка, парный. Перед элементами списка ставится порядковый номер. li-определяеи элементы списка, количество неограничено. Других тегов после ol быть не может. Но внутри li можно размещать другие. ol>li*5+Tab-создание списка из 5 строк
em
!-- <!--Текст комментария--> Тег комментария, парный. Он не отображается на экране. Можно комментировать выделенные строки кода с помощью комбинации клавиш Ctrl+K затем Ctrl+C . Для снятия комментария - повторить нажатие тех же клавиш. Комментарии можно делать в одну или несколько строк.
hr <hr align="center" color="#fff" size="1" width="480px" /> <p align="center" style="color: #fff;">Был бы двор</p> <hr align="center" color="#fff" size="1" width="480px" />
  • align - устанавливает выравнивание разделителя (right, center или left);
  • color - задает цвет линии;
  • size - определяет толщину линии;
  • width - задает ширину линии;
  • noshade - отменяет трехмерный эффект линии
 hr - горизонтальный разделитель вида ………….

HTML - Тег для создания ссылок <a href="" target="_blank" title=""> Cсылка </a>

В HTML существует единственный парный тег для создания ссылок. Он позволяет переходить по ссылкам не только внутри одного проекта, но и переходить на сторонние рессурсы или копировать файлы. В зависимости от значения атрибута href создается та или иная ссылка.

Общие атрибуты для разных видов ссылок:

  • target="_blank" - открывает рессурс в новой вкладке (для открытия в текущей вкладке - атрибут опускается)
  • title="" - выводит подсказку при наведении мышки на ссылку
  • Ссылка - выводит на экран название кнопки для ссылка

HTML - Тег <table> для форматирования таблиц

Формат Атрибуты Варианты Emmet

<table border="1" width="100%" align="center">

<thead><tr>

<th></th>...<th></th> </tr></thead>

<tbody>

<tr>

<td></td>.... <td></td>

</tr>

<tr><td></td>...<td></td>

</tr>

</tbody>

<tfoot>

<tr>

<td colspan="2"></td>

</tr>

</tfoot>

</table>

 border – границы таблицы (0-без границ, 1-границы)

width - ширина таблицы (может задаваться в % к окну или в px)

align - выравнивание (left—слева, right—справа, center-центр)

background - фоновый рисунок таблицы(background="URL")

bgcolor - цвет фона таблицы (bgcolor="цвет")

bordercolor - цвет рамки (bordercolor="цвет")

cellpadding - расстояние от рамки до текста (cellpadding="значение")

cellspacing - расстояние между внешинми границами ячеек (cellspacing="значение")

thead ..../thead - шапка таблицы, внутри готорого помещается строка с помощью тега tr../tr, а ячейки шапки внутри tr размечаются тегом th. При этом шапка печатается полужирным шрифтом.

tbody...../tbody - тело таблицы, которое состоит из строк (тег tr). А ячейки помечаются тегом (td). Количество строк и ячеек не ограничено, но в каждой строке должно быть одинаковое количество ячеек.

t-foot ..../tfoot-нижняя итоговая часть таблицы. Внутри строки (тег tr), которые разделяются на ячейки (теги td).

rowspan - объединяет ячейки по вертикали

colspan - объединять ячеек по горизонтали

tr>th*3-создание шапку из 3 граф

имя тега и Tab - быстрый набор тега HTML

tbody>tr*2>td*3 - создает тело таблицы из 2 строк и 3 граф

table>thead>(tr>th*3)^tbody>tr*2>td*3-создает таблицу с шапкой из 3 ячеек и 2 строк. ()-выполняют роль группировки, а символ ^ поднимает тег на уровень вверх в иерархии тегов.

Мой первый сайт

Я - пенсионер, с техническим образованием и 30-летним стажем работы в IT-индустрии. По роду своей деятельности мне не приходилось заниматься разработкой и созданием сайтов. Но размер пенсии не позволяет сохранить свой прежний образ жизни и поэтому я задумалась о дополнительном заработке. Возможно это только мечта, но попробовать стоит. Интернет в помощь.

К каким выводам я пришла, несколько дней просидев в Интернете?

  • Можно пойти простым путем: записаться на платные курсы. Но не верьте тем, кто говорит, что можно научиться создавать сайты за 3-5дней. За это время можно только понять, что для этого нужно и список команд кода. Если уж платить деньги, то придется выбирать в интернете по отзывам. При этом надо помнить, что не могут быть все отзывы хорошие. Если это так, то не стоит верить. Лучше всего курсы, где к вам прикрепляют наставника, который проверяет ваши работы и дает советы. Но это совсем не дешево.
  • Я выбрала более длинный: учиться по бесплатным проектам в Интернете и советам знакомых. Для этого перебрала множестно ресурсов. Есть сайты с бесплатными учебниками, в которых дается основная информация. Есть структурированные сайты, где можно найти дополнительную информацию. Есть чаты, где можно задавать вопросы. Не надо бояться показаться нелепым и смешным. Кто то посмеется и нахомит, но всегда найдется тот, кто даст дельный совет.
  • Итог: поняла, что для создания сайта (самого простого, который будет виден в Интернете) необходимо:
    • изучить несколько веб-языков (не один)
    • установить на ПК редактор кода
    • начать создавать сайт (т.к. изучать код без практики невозможно)
    • зарегистрироваться на хостинге для запуска сайта (это веб-сервер в Интернете)
    • Веб языки для создания сайта
    Название языка Назначение
    HTML предназначены для верстки сайтов (верстка - это размещение элементов сайта по нужным местам) создание структуры страниц сайта, без алгоритмов
    CSS оформление внешнего сайта и его расположение на странице
    PHP программирование сайта (например, регистрация пользователей, с передачей данных на веб-сервер)
    JavaScript задание функциональности, (например "оживить", сделать меняющиеся картинки)
    Дополнительные инструменты
    Название Назначение Ссылка на ресурс
    Браузер Удобнее всего использовать Google Chrome
    Photoshop Графический редактор (версии не ниже 2015), предназначен для верстки сайта или открытия уже готового макета
    VSCode Бесплатный редактор кодов https://code.visualstudio.com/
    Google Fonts Сервис бесплатных шрифтов (платные-предоставляет заказчик) https://fonts.google.com
    Figma Сервис с готовыми шаблонами позволяет быстро создать макет сайта https://www.figma.com/
    Zeplin Сервис для измерения размеров элементов в макете сайта (можно установить расширение в фотошот) или использовать в браузере https://zeplin.io/
    PerfectPixel Сервис для проверки правильности верстки сайта с помощью наложения его на макет. https://www.welldonecode.com/perfectpixel/
    GitHubPages Хостинг (бесплатный) для размещения готового сайта в интернете https://github.com

    1. Установить и настроить бесплатный редактор кодов VSCode Обзор редактора
    2. Изучить и написать структуру с помощью HTML Основные коды HTML
    3. Изучить работу со шрифтами Работа со шрифтами
    4. Навести красоту с помощью CSS Основы CSS
    5. Изучить Figma Работа с Figma
    6. Перенести на бесплатный Хостинг Работа с GitHubPages
    7. Добавить функционал с помощью JavaScript Основные коды JavaScript
    8. Запустить и посмотреть, что получилось
    9. Запрограммировать с помощью PHP (необязательный пункт) Основные коды PHP
    10. Изучить Zeplin
    11. Изучить PerfectPixel

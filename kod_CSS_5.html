CSS стили для горизонтальной разделительной линии

Тег Описание
width

Длина линии.

Может быть задана значением в px, em или процентах.

hr {width: 200px;}

hr {width: 50%;}
background-color

Цвет линии.

hr {background-color: #green;}
background-image

Изображение для линии.

В качестве изображения можно использовать градиент.
border: none;

Убирает тень от линии.
height

Толщина линии.

hr {height: 1px;}

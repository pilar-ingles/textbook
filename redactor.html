Обзор редактора DSCode

Для установки редактора надо перейти на его домашнюю страницу для скачивания дистрибутива https://code.visualstudio.com/ и изучения документации с подробными описаниями на английском языке.

По умолчанию VSCode запускается с англоязычным интерфейсом, но при желании можно настроить русский.

Пять значков в левой вертикальной части окна редактора позволяют открывать боковые панели:

  1. Explorer (Проводник) – файловый менеджер для работы с папками и файлами в проекте;
  2. Search (Поиск) – инструмент для поиска и замены строк в файлах проекта;
  3. Source Control (Система контроля версий) – инструмент для работы с системами контроля версий;
  4. Debug (Отладка) – средства для отладки кода и запуска программ;
  5. Extensions (Расширения) – менеджер для работы с расширениями редактора.

Настройки

В настройки можно зайти 2 способами:

  1. пункт меню File/Preferences/Settings(Файл/Настройки/Параметры)
  2. шестеренка в нижнем левом углу/Settings(Параметры).
Слева будет расположен список доступных настроек, справа – их значения и описания.

Вкладки User Settings (Параметры пользователя) и Workspace Settings (Параметры рабочей области). Вторая сохраняет настройки для текущего проекта в скрытой вложенной в проект папке под названием .vscode. Чтоыб скрыть эту папку можно при помощи настройки Files: Exclude, добавив туда строку ".vscode" (данная настройка позволяет исключить определенные файлы или папки из панели Проводника).

Для себя я отметила 2 группы настроек:

  • Для обновления редактора и сохранения файлов. Все настройки по обновлению редактора я отключила.
    • Auto Save – режим автоматического сохранения файлов (включено)
    • Auto Check Update - автообновление редактора(включено)
    • Auto Update - автообновление редактора(включено)
    • Enable Windows Back ground Update - автообновление редактора(включено)
    • Mode - автообновление редактора(включено)
    • Show Reloase Notes - автообновление редактора(включено)
  • Для настройки вида редактора
    • Font Size – размер шрифта в пикселях.
    • Font Family – название шрифта, используемого в редакторе
    • Tab Size – количество пробелов в табуляции (стандарт-два пробела)
    • Cursor Style – стиль курсора
    Настройки применяются сразу после их сохранения с помощью комбинации клавиш Ctrl+S

    Горячие клавиши

    Просмотр горячих клавиш и настройки их под себя выполняется в пункте меню File/Preferences/Keyboard Shortcuts (Файл/Параметры/Сочетание клавиш) либо через шестеренку/Keyboard Shortcuts (Сочетание клавиш)

    • Ctrl+shift+E - вызов проводника
    • Shift+Alt+F - форматирование кода (для красоты)
    • Alt+стрелка вверх/стрелка вниз – перемещение строки кода вверх/вниз.
    • Shift+Alt+стрелка вверх/стрелка вниз – дублирование строки или выделенный блок кода вниз или вверх соответственно.
    • Ctrl+Shift+K – удаление строки кода целиком
    • Tab – сдвиг кода вправо
    • Shift+Tab – сдвиг кода влево
    • Ctrl+стрелка вправо/стрелка влево – перемещение курсора к следующему/предыдущему слову.
    • Ctrl+Shift+стрелка вправо – выделение кода от позиции курсора до конца слова.
    • Ctrl+Shift+стрелка влево – снятие выделения.
    • Alt+Shift+стрелка вправо – нарастающее выделение кода (сначала выделяется слово, в котором установлен курсор, затем целая строка, затем блок и весь текст).
    • Alt+Shift+стрелка влево – снятие выделения.
    • Сворачивать и разворачивать блоки можно использовать стрелки, которые появляются при наведении указателя мышки на пустую область рядом с порядковым ноиером строки.
    • Ctrl+K затем Ctrl+C - вставить комментарий
    • Ctrl+Z - отменить последнее действие
    • F11 - полный экран
    • Alt+W - обертывание (Emmet)

    Расширения (дополнительные функции редактора)

    При переходе на данную панель отображается список уже установленных расширений (Enabled) и расширений, рекомендуемых к установке (Recommended). Для установки нажимаем на Install (Установить). После установки расширения редактор следует перезагрузить.

      Какие расширения у меня установлены

    • Emmet-помогает быстро создавать шаблоны (установлено по умолчанию). В дальнейшем, при описании тега буду указывать как его использовать с помощью Emmet(Em:символы+клавиша)
    • Vscode-lahguage-pack-ru - настраивает русский интерфейс
    • Live Server - простой веб-сервер для удобной работы со статическими и динамическими страницами. Внизу слева появляется значек GoLine. При нажатии на него рабочий код будет открыт в браузере. Он поддерживает автоматическое обновление открытых страниц при изменении их исходного кода. Для выхода из решима - нажать на значение порта:Port:
    • Auto Rename Tag – ищет закрывающую пару тега и переименовывает её.
    • Advanced New File – помогает создавать файлы быстрее при помощи сочетания клавиш Ctrl+Alt+N (если при создании файла задается относительный путь к нему, то все промежуточные каталоги будут созданы автоматически).
    • Bookmarks – позволяет добавлять в код закладки. Это очень удобно, когда приходится работать с большим количеством файлов или код пишется в большом файле сразу сверху и снизу. Закладки будут сохраняться в отдельной панели закладок.
    • HTML tag wrapper – позволяет по сочетанию клавиш оборачивать HTML-код в различные теги.
    • Image preview – показывает миниатюру изображения и его размеры в строке со ссылкой в коде
    • Indentation Level Movement – позволяет быстро перемещаться между блоками кода (например, переходить между функциями, не заходя в них).
    • Path Intellisense – расширение для автодополнения путей в файловой системе. При вводе путей редактор будет подсказывать, какую папку или файл выбрать.
    • Material Icon Theme – предоставляет красивые иконки для файлов проекта.
    • Htmltagwrap - Позволяет оборачивать выделенный текст в HTML-теги (теги вставляются по нажатию комбинации клавиш Alt+W).
    • CSS Peek - возможность редактирования переменных, классов или функций напрямую из кода, в котором они используются для стилей CSS
    • Beautify css/sass/scss/less – предназначается для форматирования кода файлов стилей (по умолчанию редактор умеет форматировать только HTML).
    • intelliSense for CSS class names – считывает CSS-файлы и предлагает классы из них во время написания верстальщиком HTML-разметки.

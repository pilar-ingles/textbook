Работа со шрифтами

При создании макета сайта дизайнер подбирает шрифт в соответствии со своим видением конечного результата. Поэтому задача верстальщика при выборе шрифта при создании сайта руководствоваться требованиям дизайна. Название шрифта можно увидеть при просмотре макета страницы в специальных программах типа Adobe Photoshop, Avocode или Figma.

Все шрифты делятся на 3 вида:

  1. Безопасные шрифты

    Безопасными являются те шрифты, которые установлены на большинстве устройств и доступны на всех платформах и операционных системах. Например, шрифты Arial Black, Arial, Comic Sans MS, Courier, Courier New, Georgia, Impact, Times New Roman, Trebuchet MS, Verdana, Webdings. Полный список безопасных шрифтов можно легко найти в Интернете.

    Если мы используем безопасный шрифт, то текст будет выглядеть одинаковым на Windows, MacOS, и на Linux, т.к. такой шрифт установлен в системы по умолчанию. Браузер найдет его и применит. Для использования такого шрифта достаточно в CSS-правилах указать в качестве значения свойства font-family название этого шрифта, например: body {font-family: Arial;}

  2. Бесплатные шрифты

    Но не все шрифты предустановлены в системы. Если нужный шрифт не найден среди безопасных, то сначала его надо поискать среди бесплатных шрифтов. Для этого надо воспользоваться сервисом Google Fonts https://fonts.google.com Данный сервис предоставляет доступ только к бесплатным общедоступным шрифтам. Например, нам нужен шрифт Open Sans. Он не является безопасных, поэтому мы должны браузеру указать его местонахождение.

    Переходим на сайт https://fonts.google.com, и с помощью поиска находим нужный нам шрифт. Заходим во вкладку шрифта и нажимаем на кнопку "Получить шрифт (get font)".

    Подключить шрифт на страницу можно двумя способами: обычным (ссылка) и с помощью директивы (@IMPORT).

    Для подключения шрифта обычным способом надо скопировать содержимое раздела "Вставьте код в head вашего HTML-кода" с помощью кнопки "Скопировать код". Вставить код в HTML-страницу в раздел HEAD до подключения основного файла стилей main.css. После этого в файл стилей нужно записать свойство font-family для загруженного шрифта body {font-family: 'Open Sans', sans-serif; }

    Альтернативный способ позволяет подключать шрифт непосредственно в CSS-файл, при этом в разделе HEAD тег link для загружаемого шрифта указывать не нужно. Выбираем "@импорт (@import)" и копируем содержимое раздела "Open Sans: CSS-класс для переменного стиля" с помощью кнопки "Скопировать код". После этого все скопированное вставляем в файл со стилями.

    Любой из этих двух способов позволяет подключить бесплатный шрифт с сервиса Google Fonts.

  3. Платные шрифты

    Если шрифта нет в безопасных и на Google Fonts, то он является платным. Дизайнеры их покупают для использования и предоставляют верстальщикам. Такие шрифты, как правило, распространяются в формате OpenType в виде набора файлов с расширением otf (шрифты, основанные на PostScript) или ttf (шрифты, основанные на TrueType). Каждый такой файл соответствует определенному начертанию символов (обычный, наклонный, полужирный и т. д.).

    Шрифты в формате OpenType можно устанавливать в операционной системе, но для использования в Интернете они не оптимизированы, т.к. имеют большой размер. Поэтому их сначала надо конвертировать в формат, который будет пригоден для веб-разработки. Для этого существуют специальные онлайн-сервисы, например Font Squirrel https://www.fontsquirrel.com.

    Для конвертации заходим на вкладку Generator, нажимаем на кнопку UPLOAD FONTS и выберем файлы с нужными начертаниями для конвертации. Пусть это будут файлы для обычного текста (Roman), жирного начертания (Bold), наклонного (Italic) и жирного наклонного (Bold Italic) текста.

    После загрузки файлов со шрифтами нужно необходимо настроить параметры конвертации с помощью режима EXPERT. Для современных браузеров достаточно преобразовать шрифты к форматам WOFF и WOFF2. Также в параметрах конвертации можно указать таблицы символов, которые будут включены в данный шрифт. Для этого в разделе Subsetting выбираем пункт Custom Subsetting и устанавливаем флажки на нужных языках и наборах символов. Но надо помнить, что от количества вариантов символов зависит размер итоговых файлов для шрифта. Чем они больше, тем дольше будет загружаться страница браузера. Перед тем как генерировать, можно посмотреть какие символы будут сгенерированы в результате конвертации шрифта. Соглашаемся с условиями предоставления сервиса и нажимаем на кнопку DOWNLOAD YOUR KIT. В результате все файлы со шрифтами в веб-формате будут загружены одним zip-архивом на локальный ПК.

    Из всего списка файлов понадобятся все файлы с расширениями woff и woff2, которые мы скопируем в папку fonts в нашем проекте, и файл stylesheet.css, содержимое которого надо скопировать в файл main.css, подключенный к основной странице index.html. В файле main.css указываются корректные пути до файлов шрифта. После этого шрифты уже можно использовать в CSS-стилях.

    Если в свойстве font-family указывается небезопасный шрифт, то через запятую нужно указать так называемые fallback-шрифты, которые будут использованы при невозможности подключения основного шрифта. Такие резервные шрифты должны быть безопасными, чтобы они наверняка могли быть загружены на компьютере пользователя. Также после fallback-шрифтов через запятую нужно указать, к какому семейству относится подключаемый шрифт: sans-serif (шрифт без засечек) или serif (шрифт с засечками). Таким образом, браузер сначала попытается загрузить платный шрифт. Если это не удастся, то будет подключен fallback-шрифт. Если же и этот шрифт подключить не получится, то будет использован любой шрифт без засечек, доступный на устройстве, где загружается страница.

Наследуемые свойства

Все свойства, предназначенные для стилизации шрифтов, являются наследуемыми. Если такое свойство определено для элемента, то оно сохранится по умолчанию для его потомков. Чтобы изменить значение свойства, полученного по наследству, нужно его перезаписать для нужного блока. Также свойством наследования обладают свойства font-size, font-weight, text-decoration, line-height, font-family, font-style, text-align, text-transform, text-indent, letter-spacing, word-spacing, white-space, list-style и некоторые другие специфические свойства, связанные с текстом.

Общие стили текста

Так как свойства, связанные со стилизацией текста, являются наследуемыми, то общие настройки текста нужно делать на корневом теге body. Здесь можно указать значения свойств font-family, font-weight, font-size для шрифта, который наиболее часто используются на верстаемой странице.

Например: Указав размер шрифта (свойство font-size), нужно задать и значение межстрочного интервала (свойство line-height). Хотя значения свойства line-height можно задавать в пикселях, делать этого не следует: лучше использовать относительные единицы измерения em. Напомним, что один em равен размеру шрифта на элементе (если для этого элемента явно задано свойство font-size) или его родителе. Поэтому межстрочный интервал, заданные в em, будет пропорционален высоте букв: чем больше размер шрифта, тем шире будет интервал между строк. Значение для line-height также можно записать просто в виде числа, без указания единиц измерения. Это число будет воспринято браузером как коэффициент, на который нужно умножить размер шрифта, чтобы получить значение межстрочного расстояния. Обычно для line-height=1.42 (при этом размеру шрифта в 16 пикселей будет соответствовать межстрочный интервал в 20 пикселей). body {font-family: 'helveticaneue', Helvetica, sans-serif; font-weight: 300;} Если указанное значение свойства line-height будет меньше 1, то строки будут наползать друг на друга. Также в правила для тега body можно поместить свойства, отвечающие за сглаживание шрифтов на экране.

Составное свойство font позволяет задавать сразу несколько настроек шрифта. При использовании этого свойства обязательно нужно указать размер и семейство шрифта: font: 16px 'helveticaneue', Helvetica, sans-serif; Дополнительно в font можно указать значения параметров font-weight, font-style и line-height (последнее свойство записывается через слеш после размера шрифта). Например: font: 300 16px/1.42 'helveticaneue', Helvetica, sans-serif;

<form action="" method="" enctype="" autocomplete="" >

Теги ввода данных

Тег отправки данных на сервер

</form>

 action – Адрес страницы на веб-сервере, на которую будут посланы данные из формы.

method – Метод передачи данных на сервер. По умолчанию "method="get". Но существуют и другие, например POST.

enctype – Способ, которым будут закодированы данные из формы. Если форма загружает на сервер файлы, то для тега

нужно установить атрибут enctype="multipart/form-data"

autocomplete – Включение или отключение режима автозаполнения полей формы значениями, которые вводились ранее. Значения "on" или "off" ("on"-по умолчанию)

Cоздает формы для ввода данных для передачи их на сервер

Задача верстальщика – привести внешний вид формы в соответствие с предоставленным макетом, а значения атрибутов action, method и enctype должен предоставить разработчик, отвечающий за серверную часть приложения, обрабатывающего данные из формы. Более полные проверки отправляемых на сервер данных делают с помощью JavaScript.
Теги ввода данных.
Общие атрибуты: name-Идентификация данных, передаваемых из формы

required-Обязательность заполнения данных в нужном формате перед отправкой. При пустом или неправильно заполненном по типу поле при попытке отправления формы будет выведена подсказка (внешний вид зависит браузера).

autofocus-Автоматическая установка курсора в определённое поле формы сразу после загрузки страницы. При этом атрибут в теге ввода "tabindex"–задает порядок перехода между полями формы по нажатию клавиши. Переход между полями осуществляется клавишей Tab. Переход в предыдущее поле-по нажатию сочетания клавиш Shift+Tab

input type="file" - Выбор файлов для последующей их загрузки на сервер

На странице при этом отобразится кнопка для выбора файла и будет написано имя уже выбранного файла (внешний вид кнопки и надпись на ней зависят от браузера). После нажатия на эту кнопку откроется стандартное окно ОС для выбора файла
<input type="range" min="0" max="100" step="10"> min-минимальная граница max-максимальная step- дискретный шаг, с которым перемещается ползунок слайдера
input type="range" - Выводит на страницу слайдер для выбора числовых значений из диапазона. Сами граничные значения диапазона Нужно верстать на странице отдельно. Если нужно динамически отображать текущее значение диапазона, на котором находится ползунок, то придется применять сценарии JavaScript..
<input type="number" min="0" max="50" step="5"> min-минимальная граница max-максимальная step- для задания допустимого диапазона значений и шага, с которым будет меняться значение в поле ввода.
input type="number" - Ввод числовых значений, которые можно динамически увеличивать или уменьшать с помощью кнопок со стрелками в конце поля ввода.
input type="hidden" input type="hidden"- Ничего не выводит на страницу, но сохраняет указанные в нём данные для дальнейшей отправки их серверу. Используется этот тег чисто в технических целях для скрытой передачи серверу служебной информации (например, id товара, который был выбран в форме).
